Perform a read transaction on the bus.

This function will perform a read on the APB bus. It will start reading from the address and continue for the number passed in with burstCount. With this updating the address and placing the data in the given buffer.

Parameters
addressStart address to read from
bufferDatabuffer to store the data read
burstCountNumber of transactions in this burst

Perform a burst write transaction on the bus.

This function will perform a write on the APB bus. It will write the first data element to the passed address and then continue to write the data in the buffer in the following addresses. Which will continue for the amount in burstCount.

Parameters
addressStart address of the burst transaction
bufferDatabuffer that holds the data to write
burstCountNumber of transactions in this burst
