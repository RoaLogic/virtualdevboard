Detailed Description

VGA virtual development board component.

Author
Bjorn Schouteten

This class communicates with the verilator component, any class that wants to listen to it should register itself through the subject-observer pattern.

This class sents one event, the vgaDataReady event. It passes data according to the sVgaData structure. Which passes the number of horizontal and vertical lines. The data array is build up as a single array with a size of horizontal lines * vertical lines. All the horizontal lines are appended after each other.
