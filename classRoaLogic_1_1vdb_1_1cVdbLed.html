Detailed Description

Virtual GUI LED controlled by verilog instance.

Author
Bjorn Schouteten
Version
0.1
Date
13-oct-2024

This class controls a verilated LED instance.

It takes the verilator event through the verilator callback and notifies anyone listening to this led. This class fully runs in the verilated context. The base is the cVDBCommon class which does all the low level handling and setting up the callback mechanism for any DPI functions. The user shall call the cVDBCommon::processVerilatorEvent with the scope and the eventual event.
