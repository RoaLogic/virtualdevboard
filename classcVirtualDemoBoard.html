Detailed Description

WX widgets virtual demo board implementation.

Author
Bjorn Schouteten
Version
0.1
Date
19-okt-2024

This class is a GUI implementation for the virtual demo board

It is specificly coupled with wxWidgets and shall only use wxWidget elements.

to communicate with the verilated design it inherits from the cGuiInterface, since this interface is a subject we can notify events.

Attention
The virtual functions coming from the cGuiInterface are all run within the verilator context, so we must make sure that we don't change any GUI elements.
