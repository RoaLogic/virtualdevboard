Detailed Description

LED virtual development board component.

Author
Bjorn Schouteten, Richard Herveille

This class subscribes to it's VDB component and awaits any events. Corresponding events are received in the notify function, which runs in the verilated context. GUI adjustments have to be done in the GUI thread, so a wxEvent is sent and following the thread switch then processed in the onLedEvent function.

The LED is drawn in the OnPaint function, which also defines how the LED looks like
