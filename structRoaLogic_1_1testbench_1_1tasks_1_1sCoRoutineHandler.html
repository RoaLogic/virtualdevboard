The await suspend function for the awaitable type.

This function suspends the coroutine and implements the needed logic. It stores the passed coroutine handle into the waitingCoroutine handle of our current promise type.

This can then be used to resume the coroutine that called co_await

template<typename T >
T RoaLogic::testbench::tasks::sCoRoutineHandler< T >::getValue ( )
inline

Get the Result.

This function returns the value gotten from the promise type. Or easier said, it returns the result of the coroutine. This can either be the result from co_yield as well as co_return.

Attention
When final_suspend returns suspend_always we cannot get the value of the co_return expr. This value will then be changed, due that the promise is already deleted.
Returns
T Returns coroutine value or false when there is no value
