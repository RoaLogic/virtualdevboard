This class is the observer implementation of the observer pattern.

A class shall derive from this class and implement the notify() function, it then subscribes itself to a subject. When the subject now has a event it will call the notify function and the event can be processed in any way.

Attention
This function will always run in the thread context of the calling thread. So changing variables of another thread must be made thread safe within the notify() function implementation.
